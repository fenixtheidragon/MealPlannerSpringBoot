/api/meals

example of body (full Ingredient):
{
    "name": "Apple",
    "amountType": "GRAMS",
    "availableAmount": 1000
}
amountType possible values: grams, kilograms, pieces, litres

1)GET - возвращает список приёмов пищи в виде List
Requests nothing
Returns body (ResponseEntity<List<Meal>>) and HTTPStatus(OK)

2)POST - сохраняет один приём пищи и возвращает сохраненный
Requests body (Meal)
Returns body (ResponseEntity<Meal>) and HTTPStatus(CONFLICT/OK)

3)PUT - обновляет информацию по одному приёму пищи
Requests body (Meal)
Returns body (ResponseEntity<Meal>) and HTTPStatus(NOT_FOUND/CONFLICT/OK)

4)DELETE - удаляет один приём пищи по id
Requests param (Long id) (example: /meals?id=1)
Returns HTTPStatus(NOT_FOUND/OK)

/api/ingredients/{id}

GET - находит один приём пищи по id
Requests PathVariable (Long id) (example: /meals/1)
Returns body (ResponseEntity<Meal>) and HTTPStatus(NOT_FOUND/FOUND)

/api/meals/names/{name}

GET - находит один приём пищи по имени
Request PathVariable (String name) (example: /meals/Apple)
Returns body (ResponseEntity<Meal>) and HTTPStatus(NOT_FOUND/FOUND)
